# 📘 Python Data Science Notes

## 🚀 Overview

This repository contains well-structured notebooks covering essential concepts of Data Science using Python libraries. It is designed for learners who want hands-on practice with real examples.

---

## 🎯 Objectives

* Understand core Data Science libraries
* Practice coding with real examples
* Build a strong foundation for projects
* Prepare for interviews

---

## 📚 Topics Covered

### 🔢 NumPy

* Array creation and operations
* Indexing and slicing
* Mathematical functions
* Broadcasting

### 📊 Pandas

* DataFrames and Series
* Data cleaning
* Filtering and grouping
* Data analysis

### 🧠 OOP in Python

* Classes and Objects
* Inheritance
* Polymorphism
* Encapsulation

### 📈 Data Visualization

* Matplotlib basics
* Seaborn plots
* Graph customization
